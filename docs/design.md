# Design

## 目的

`worm-game-flat` は、Godot の機能一覧を学ぶためのデモではありません。

小さな Worm ゲームを完成させる過程で、Godot のプロジェクト、Scene、Node、GDScript、入力、時間、描画、状態管理に自然に触れることを目的とします。

## 視覚表現

画面を `N × N` のセルに分け、各セルを円として描きます。

| 要素 | 表現 |
|---|---|
| 地面 | 基本色の円 |
| 頭 | 頭色の円 |
| 方向 | 頭の円周に内接する小円 |
| 体 | 体色の円 |
| 餌 | 餌色の円 |

頭の大円の半径を `R`、方向を示す小円の半径を `r` とすると、小円の中心は頭の中心から方向ベクトルへ `R - r` だけ移動します。これにより小円は大円の内側で円周に接します。

```text
marker_center = head_center + direction * (head_radius - marker_radius)
```

## ゲームルール

- 起動するとワームは右へ自動移動する
- `h` `j` `k` `l` で進行方向を変更する
- 真後ろへの方向転換は受け付けない
- 壁または自身に接触すると終了する
- 餌を食べると体が一つ伸びる
- 終了後は `r` で再開できる
- `Esc` で終了する

## 意図的に設けないもの

最小形では、タイトル画面、設定画面、チュートリアル表示、スコア、音、アニメーション、Steam連携を設けません。

それらは、このゲームの構造を理解した後に追加できる発展要素です。
